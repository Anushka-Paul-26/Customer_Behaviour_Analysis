# Customer_Behaviour_Analysis

An end-to-end Data Analytics project focused on analyzing customer shopping patterns and transforming raw transactional data into meaningful business insights using **Python, PostgreSQL, and Power BI**.

This project simulates a complete real-world analytics workflow — from data cleaning and SQL-based business analysis to interactive dashboard reporting for business decision-making.

---

## Project Overview

The objective of this project is to analyze customer purchase behavior and uncover patterns related to:

* customer spending habits
* subscription behavior
* category-wise sales performance
* age-group based purchasing trends
* payment preferences
* shipping impact on customer satisfaction

Using Python for preprocessing, SQL for business analysis, and Power BI for dashboarding, this project demonstrates how raw customer data can be transformed into actionable insights.

---

## Problem Statement

Retail businesses generate large volumes of customer transaction data every day. However, raw transactional data alone is difficult to interpret and does not directly support strategic decisions.

This project aims to solve that by answering key business questions such as:

* Which customer groups generate the most revenue?
* Which product categories perform best?
* How does subscription status affect customer behavior?
* Which age groups contribute the most to sales?
* What patterns influence customer purchase decisions?

The goal is to convert raw customer transaction records into a clear, interactive business intelligence dashboard.

## Project Workflow

### 1. Data Cleaning & Preprocessing (Python)

Performed data cleaning and transformation on raw customer shopping data.

Tasks completed:

* handled missing values
* standardized column names
* removed unnecessary columns
* created customer age groups
* converted purchase frequency into numeric format
* prepared cleaned dataset for SQL and Power BI

---

### 2. Exploratory Data Analysis (Python)

Performed exploratory analysis to identify customer behavior patterns.

Key analysis included:

* customer spending trends
* revenue distribution across categories
* age-group based purchase patterns
* shipping and subscription behavior
* customer review trends

---

### 3. Business Analysis (SQL)

Used PostgreSQL to answer business-focused analytical questions.

SQL analysis covered:

* revenue by gender
* revenue by category
* revenue by location
* subscription vs customer loyalty
* payment method usage
* shipping type vs customer satisfaction
* category performance vs customer rating

---

### 4. Dashboard Visualization (Power BI)

Built an interactive Power BI dashboard to present business insights in a clean and stakeholder-friendly format.

The dashboard includes:

* **KPI Cards**

  * No of Customers
  * Average Purchase Amount
  * Average Review Rating

* **Interactive Filters**

  * Subscription Status
  * Gender
  * Category
  * Shipping Type

* **Visual Insights**

  * Percentage of Customers by Subscription Status
  * Revenue by Category
  * Sales by Category
  * Revenue by Age Group
  * Sales by Age Group

This dashboard enables quick analysis of customer behavior, spending trends, and segment-wise performance.
